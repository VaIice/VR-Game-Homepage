## ⚙ 프로젝트 개발 배경
- 코로나 시절 VR 기기 보급의 증가
- 쉽게 접하기 힘든 VR 기술에 도전
- VR기기 플레이에 있어 새로운 즐거움 창출
- 시장에 직접 뛰어들어 창의적 콘텐츠를 게임에 투영
- 일상의 무료함에서 벗어가 VR기기를 통한 매력적인 게임 체험을 제공
- 직접 제작한 홈페이지로 인한 사용자들과의 상호작용 가능
- 게시판 등의 소셜 경험을 촉진하여 전반적인 유지율 증가
